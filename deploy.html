Netlify 部署指南

方法 1：拖放部署（最快）

  1. 访问 https://app.netlify.com/drop
  2. 拖放 prompt-templates-mvp 文件夹到浏览器
  3. Netlify 会自动部署
  4. 获得一个免费 URL（如 prompt-master-xyz.netlify.app）

方法 2：Netlify CLI

npm install -g netlify-cli
netlify deploy --dir=prompt-templates-mvp --prod

方法 3：GitHub Pages

  1. 创建 GitHub 仓库并推送代码
  2. 在仓库 Settings → Pages 中启用 GitHub Pages
  3. 选择 main 分支作为源
  4. 访问 https://[username].github.io/prompt-templates-mvp/

推荐：Netlify 拖放

理由：

  • 无需安装 CLI
  • 无需登录
  • 部署速度最快（秒级）
  • 自动 HTTPS
  • 支持自定义域名

下一步
